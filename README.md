## Project Overview

This project involved developing an intelligent agent for the Othello board game using adversarial search techniques such as Minimax and Alpha-Beta Pruning.

The objective was to design an agent capable of evaluating board states effectively and making strategic decisions against different opponents.

## My Contributions

* Designed and implemented an advanced heuristic evaluation function for board-state scoring
* Combined multiple evaluation metrics, including piece difference, mobility, and positional advantage
* Applied position-based weighting to prioritize strategic cells such as corners and edges while penalizing risky near-corner positions
* Developed a dynamic weighting strategy that adapts evaluation priorities across early, mid, and late game phases
* Improved agent decision quality by balancing offensive and defensive gameplay strategies
